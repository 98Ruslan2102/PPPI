Тема проекта

3d редактор, с реализацией не менее 20-ти программных средств доступных в 3ds Max / Maya / Cinema 4D .

Автор проекта

Самойленко Руслан

Результаты работы по предмету ПППИ:

№1 Базовые принципы работы с системами контроля версий
№2 Работа с ветками
№3 Восстановление данных из ранних ревизий
№4 Создание самодокументирующегося кода
№5 Разработка ручной документации
№6 Веб-сайт из репозитория
